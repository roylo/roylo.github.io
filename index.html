AngularJs Learning Note-2

Jul 9th, 2014 | Comments

Filter

  • use pipe “|”
    • the following example will give price the dolloar sign(localized), Specifies number of decimals
  • Formatting with filters
    • Our Recipe {{ data | filter:options }}
    • date example {{'1388123412323' | date:'MM/dd/yyyy @ h:mma'}} => 12/27/2013 @ 12:50AM
    • uppercase & lowercase `{{‘octagon gem’ | uppercase}} => OCTAGON GEM
    • limitTo {{'My Description' | limitTo:8}} => My Descr
      <li ng-repeat="product in store.products | limitTo:3">
    • orderBy (descending order) <li ng-repeat="product in store.products | orderBy:'-price'">

images

  • Do not use <img src="{{product.images[0].full}}"/>
    • the browser tries to load the image before the Expression evaluates
  • Use <img ng-src="{{product.images[0].full}}"/>

tabs

  • When ng-click changes the value of tab
    • the {{tab}} expression auto gets updated.
  • Expressions define a 2-way Data binding
    • means Expressions are re-evaluated when a property changes.

  • How to set initial value of tab?

    • <section ng-init="tab = ">

  • ng-class directive

    • Expression to evaluate, if true, set class to “active”, otherwise nothing.
    • <li ng-class="{ active:tab === 1}">
