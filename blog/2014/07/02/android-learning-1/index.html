Android 相關

  • acore: android core
  • onXXXX function 都是 callback function, 僅會由 android 來呼叫
  • ViewGroup class
    • 處理 layout
    • 使用相對的方式去做大小設定

使用者介面實作

  • xxx.xml

    • lowercase
    • no special character
    • number cna’t be leading

  • layout 大小

    • match parent (大)
      • 填滿 parent
    • wrap content (小)
      • 只要大小可以把內容包起來就好
